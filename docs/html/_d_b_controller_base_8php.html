Detailed Description

Controller class for DB Objectify Base Controller Class. The purpose of this class is to help generate html forms and handle posts from pages back based on Objectify Model and DB Objects. The end goal is to speed up page generation and let one just focus on the logic required. Input types will try to use latest allowed by browser accessing the page.
