1 # DB Objectify
2 Tool that converts database tables into objects and creates models and controllers to work with data. Based on CodeIgniter and my_model
3 
4 Purpose of system is to generate php classes for database tables specified with models and controllers.
5 CodeIgniters configured database is used as source to work from
6 Uses MY_Model (https://github.com/lonnieezell/my_model) in Models generated.
7 
8 Models and Controllers are generated for the objects as required.
9 
10 Linked Models (one master table, multiple children) can also be generated based on config file.
11 
12 Sample Controller with display and manipulate functions can also be generated if required.
13 
14 Base classes are generated for each Object and Controller to help speed up development of DB driven systems.
